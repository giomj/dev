# conceptual-engineering-references

Curated data references for the Conceptual Engineering study track.
Companion to the /Users/james/Downloads/project/conceptual_engineering/references folder
and the Grand Council reformation sessions.

## Sources

- **n2yo** — real-time satellite tracking, TLE data, passes, radio-frequency data
- **satdump** — open-source satellite data-processing (LEO weather sats, decoders, radios)
- **gogpt** — Global Oil & Gas Plant Tracker (Global Energy Monitor) — plant-by-plant capacity, status, geographic

Each `sources/<name>/` holds: `catalog.md` (structure), `endpoints.md` (API/schema), `samples/`.
